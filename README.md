# 倍樂生台灣｜庫存儀表板

供商品部門與行銷部門主管每月開會使用的庫存健康度分析系統。
以單一 HTML 檔案運作，上傳 Excel 後即可在瀏覽器內瀏覽，資料不會上傳至任何伺服器。

## 開發目的

- **避免斷貨**：提前掌握熱銷商品的庫存消化速度，判斷是否需要追加製作
- **避免庫存處分損失**：及早發現滯銷與有庫無銷品項，討論促銷或庫存處分計畫

## 使用方式

1. 用瀏覽器開啟 `index.html`
2. 點擊「上傳最新 Excel」，選擇最新的 `●庫存查詢.xlsx`（ERP 每月中旬更新）
3. 使用頁籤瀏覽各分析頁面
4. 點擊「💾 另存更新後的 HTML」，可將含資料的儀表板另存為獨立檔案供分享

## 儀表板結構

| 頁面 | 內容 |
|------|------|
| P1 | 主力學習商品（9項）總覽：庫存量、前三個月月均銷量、A/B/C 預計完售日期；一年內可能完售者顯示警示字卡 |
| P2–P10 | 各學習商品詳細頁：近12個月銷售趨勢圖＋A/B/C預測、庫存趨勢圖、以往製作數與入庫日、前一年累積銷售數 |
| P11 | 前一年銷售超過100個的學習商品以外品項：庫存量與 A/B/C 完售預測 |
| P12 | 有庫無銷一覽：近6個月零銷售但仍有庫存的品項 |
| P13 | 滯銷篩選器：依庫存金額（10萬／25萬／50萬／100萬以上）與完售月數（6／9／12／24個月以上）交叉篩選 |

## A/B/C 銷售情境說明

| 情境 | 計算方式 |
|------|---------|
| A | 維持近三個月平均出貨數量持續銷售 |
| B | 近三個月平均出貨數量 × 120% |
| C | 近三個月平均出貨數量 × 135% |

## 資料來源

使用 ERP 匯出的 Excel 檔案，每月約 15 日更新。儀表板讀取以下工作表：

- `銷貨數 (總數)(claude)`：各月出貨明細
- `學習商品(claude)`：學習商品主檔與製作前置時間設定

## 技術說明

- 純 HTML／CSS／JavaScript，無需任何建置步驟
- [SheetJS](https://sheetjs.com/)：Excel 解析
- [Chart.js](https://www.chartjs.org/)：趨勢圖表
